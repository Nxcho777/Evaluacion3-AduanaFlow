# Prototipo funcional - AduanaFlow

## Herramienta utilizada

La herramienta seleccionada para el prototipado fue **Balsamiq**, debido a que permite representar pantallas funcionales tipo wireframe de forma rápida, clara y ordenada.

## Enlace al prototipo navegable

https://balsamiq.cloud/s765klv/pz9puem/rA310

## Pantallas consideradas

- Pantalla de inicio.
- Inicio de sesión.
- Registro de usuario.
- Dashboard principal.
- Ingreso de documentación.
- Consulta de estado de trámite.
- Panel de funcionario.
- Panel de administrador.
- Reportes.
- Notificaciones y mensajes de validación.

## Justificación

Balsamiq permite validar tempranamente la estructura del sistema AduanaFlow, enfocándose en la navegación, la distribución de información y la experiencia de usuario antes de desarrollar la plataforma final.
