## Descripción general

Este repositorio contiene los entregables asociados a la Evaluación Formativa N°3 del proyecto AduanaFlow.  
El objetivo es evidenciar el desarrollo del prototipo funcional, la aplicación de criterios de calidad basados en ISO/IEC 25000, los casos de prueba y el control de versiones utilizado durante la iteración del proyecto.
