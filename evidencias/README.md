# Evidencias

En esta carpeta se pueden incorporar capturas de pantalla del prototipo, pruebas realizadas, mensajes de validación, commits del repositorio o cualquier imagen que respalde el trabajo desarrollado.

Ejemplos de evidencias sugeridas:

- Captura del prototipo en Balsamiq.
- Captura de pantalla del dashboard.
- Captura de pantalla de formulario de registro.
- Captura de pantalla de casos de prueba.
- Captura del historial de commits en GitHub.
